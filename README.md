# Cursor IDE Setup and Exploration

## Overview

This project was completed as part of a portfolio-based assessment to demonstrate the ability to learn and work with new tools independently. The focus was on installing development tools, exploring their features, and documenting the process clearly.

## Tools Installed

* Cursor IDE
* Claude Code Extension (Anthropic)
* Codex Extension
* GitHub

## Steps Followed

1. Installed Cursor IDE from the official website.
2. Explored the interface and basic functionalities of Cursor.
3. Installed the Claude Code extension by Anthropic.
4. Installed the Codex extension.
5. Logged into the required tools.
6. Created a public GitHub repository.
7. Created and structured the README.md file.

## Challenges Faced

* Initial confusion due to multiple Claude related extensions available.
* Minor uncertainty while selecting the most relevant extension.

## How I Resolved Them

* Selected the official extension by Anthropic based on credibility and downloads.
* Explored descriptions and interface to confirm correct setup.

## Key Learnings

* Learned how to set up and explore Cursor IDE.
* Understood how to install and manage extensions.
* Gained basic knowledge of GitHub and repositories.
* Improved ability to learn tools independently.

## Conclusion

This project helped in understanding new tools quickly and documenting the process in a structured way.
